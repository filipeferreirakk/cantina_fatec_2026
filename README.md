# Sistema Cantina Fatec 2026

Este projeto é um sistema de gestão para cantinas universitárias desenvolvido para a disciplina de Estrutura de Dados. O software gerencia o estoque utilizando o conceito de Fila (FIFO), garantindo que produtos com data de entrada mais antiga sejam vendidos primeiro. Além disso, o sistema registra vendas, controla pagamentos e utiliza a biblioteca Faker para simulação de dados e Pickle para persistência em arquivo binário.

Para rodar o projeto, primeiro certifique-se de que possui o Python instalado em sua máquina. Clone este repositório e, dentro da pasta do projeto, crie um ambiente virtual com o comando `python -m venv venv`. Ative o ambiente virtual utilizando `venv\Scripts\activate` no Windows ou `source venv/bin/activate` no Linux/Mac. Com o ambiente ativo, instale as dependências necessárias executando `pip install -r requirements.txt`. 

Após a instalação, você pode iniciar o sistema de duas formas: no Windows, basta dar um duplo clique no arquivo `ABRIR_CANTINA.bat` localizado na raiz da pasta; em qualquer sistema operacional, você pode executar o comando `python main.py` diretamente no terminal. O menu interativo aparecerá na tela e você poderá navegar pelas opções de simulação, cadastro, vendas e relatórios utilizando o teclado. Ao encerrar, utilize a opção de salvar para garantir que as informações sejam gravadas no arquivo dados.dat.
